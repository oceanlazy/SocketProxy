# Python Socket Proxy
Proxy passing through itself traffic. The following modules were used to implement:
  - http.server
  - socket
  - socketserver
### Installation
```sh
$ git clone https://github.com/vadimk2016/socket_web_proxy.git
```
### Usage
Set as default proxy for your browser, then run:
```sh
$ python proxy.py
```
